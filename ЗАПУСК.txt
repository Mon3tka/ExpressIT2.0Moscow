Как запустить сайт
==================

1. Откройте терминал в папке сайта (C:\Users\Mon3tka\Desktop\site).

2. Установите зависимости (один раз):
   npm install

3. Запустите сервер разработки:
   npm run dev

   Или, если порт 3000 занят:
   npm run start

4. В браузере откройте адрес из терминала, например:
   http://localhost:3000
   или
   http://localhost:5173

5. Если не открывается — проверьте, что в терминале нет красных ошибок.
   Антивирус иногда блокирует порты: попробуйте временно отключить или добавить папку в исключения.

6. Сборка для размещения на хостинге:
   npm run build
   Папка dist — готовый сайт. Для просмотра: npm run preview
