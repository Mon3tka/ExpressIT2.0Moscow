Как запустить сайт
==================

1. Откройте терминал в папке сайта (C:\Users\Mon3tka\Desktop\site).

2. Установите зависимости (один раз):
   npm install

3. Режим разработки (только фронтенд):
   npm run dev
   Сайт: http://localhost:3000

   Полный стек (фронт + API) локально:
   npx vercel dev
   Сайт и API: http://localhost:3000

4. Деплой на Vercel:
   - Подключите репозиторий к Vercel
   - В Vercel Dashboard → Storage → Add Database → Upstash Redis
   - Создайте базу, переменные UPSTASH_REDIS_REST_URL и UPSTASH_REDIS_REST_TOKEN добавятся автоматически
   - В Settings → Environment Variables добавьте (опционально):
     ADMIN_PASSWORD=ваш_пароль
     JWT_SECRET=секрет_для_jwt
   - Deploy

5. Админка: /admin (пароль по умолчанию admin123)
