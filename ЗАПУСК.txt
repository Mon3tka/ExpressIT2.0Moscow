Как запустить сайт
==================

1. Откройте терминал в папке сайта (C:\Users\Mon3tka\Desktop\site).

2. Установите зависимости (один раз):
   npm install

3. Режим разработки (бэкенд + фронтенд вместе):
   npm run dev

   Сайт: http://localhost:3000
   API:  http://localhost:3001
   Админка: http://localhost:3000/admin (пароль по умолчанию: admin123)

4. Продакшен (одна команда после сборки):
   npm run build
   npm run start

   Сайт и API: http://localhost:3001
   Админка: http://localhost:3001/admin

5. Опционально — переменные окружения (.env):
   ADMIN_PASSWORD=ваш_пароль
   JWT_SECRET=секрет_для_jwt
   PORT=3001

6. Антивирус может блокировать порты — добавьте папку в исключения при проблемах.
